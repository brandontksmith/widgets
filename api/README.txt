Setup Instructors for Backend API

Dependencies:

	- Mongo DB
	- NodeJS

Run npm init to install the required node_modules

Setup the mongo db and populate with data by running node setup.js

You can start the server on port 3000 by running npm start

(note: be sure to change the URL on the app.js for the front-end to point to your URL or localhost... currently points to brandontksmith.com)
